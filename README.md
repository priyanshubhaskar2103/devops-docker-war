# Dockerized Java WAR Application Deployment on AWS EC2

## 📌 Project Overview

This project demonstrates how to dockerize a Java WAR application, push it to Docker Hub, and deploy it on an AWS EC2 instance.

## 🚀 Tech Stack

* Java (WAR Project)
* Apache Tomcat
* Docker
* AWS EC2

## 🐳 Docker Steps

1. Created Dockerfile using Tomcat
2. Added WAR file
3. Built Docker image
4. Pushed to Docker Hub

## ☁️ AWS Deployment

1. Launched EC2 instance
2. Installed Docker
3. Pulled image from Docker Hub
4. Ran container on port 8080
